


# Todo App

This is a simple todo app that we can add new tasks, mark as complete or incomplete or delete tasks.

## Installation

```bash
composer update
```

```bash
npm install
```
* Please Edit database name and credentials on .env before running the migrations
```bash
php artisan migrate
```

```bash
npm run dev
```
```bash
php artisan serve
```
